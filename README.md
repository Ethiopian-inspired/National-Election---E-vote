# National-Election---E-vote
<h1 align="center">🗳️ National Election E-Vote</h1>

<p align="center">
A secure web-based voting system built with <b>Django</b> that enables transparent and reliable digital elections.
</p>

<hr>

<h2>🚀 Features</h2>

<ul>
<li>🔐 <b>User Authentication System</b></li>
<li>👤 <b>Voter Registration</b></li>
<li>🏛 <b>Party / Candidate Management</b></li>
<li>✅ <b>Admin Approval Workflow</b></li>
<li>🔑 <b>Secure Token-Based Approval System</b></li>
<li>📊 <b>Election Participation System</b></li>
<li>🗂 <b>Document Upload (PDF Party Information)</b></li>
<li>🛡 <b>Role-Based Access Control</b></li>
</ul>

<h2>⚙️ Installation</h2>

<h3>1. Clone the repository</h3>

<pre>
git clone https://github.com/AlazarBekele/National-Election-E-Vote.git
</pre>

<h3>2. Go into the project folder</h3>

<pre>
cd National-Election-E-Vote
</pre>

<h3>3. Create virtual environment</h3>

<pre>
python -m venv venv
</pre>

<h3>4. Activate environment</h3>

<p><b>Mac / Linux</b></p>

<pre>
source venv/bin/activate
</pre>

<p><b>Windows</b></p>

<pre>
venv\Scripts\activate
</pre>

<h3>5. Install dependencies</h3>

<pre>
pip install -r requirements.txt
</pre>

<h3>6. Run migrations</h3>

<pre>
python manage.py migrate
</pre>

<h3>7. Run the server</h3>

<pre>
python manage.py runserver
</pre>
